介紹

從 Hugo 0.60 版本開始，默認的 Markdown renderer 已更改為 goldmark。 在 goldmark 中，默認是不呈現原始 HTML 標籤。 所以，你可能會遇到這樣的問題：

如果你需要使用到 html tags，請跟著以下的步驟 ：

我們開始吧 ！

步驟 1. 修改 config.toml

不需要安裝任何套件，只需要添加以下訊息至 config.toml 檔案並且 enable unsafe :

[markup]
  [markup.goldmark]
    [markup.goldmark.extensions]
      definitionList = true
      footnote = true
      linkify = true
      strikethrough = true
      table = true
      taskList = true
      typographer = true
    [markup.goldmark.parser]
      autoHeadingID = true
      autoHeadingIDType = 'github'
      [markup.goldmark.parser.attribute]
        block = false
        title = true
    [markup.goldmark.renderer]
      hardWraps = false
      unsafe = true
      xhtml = false

步驟 2. 驗證設置成功

儲存 config.toml 檔案後，你應該會看到執行成功。
