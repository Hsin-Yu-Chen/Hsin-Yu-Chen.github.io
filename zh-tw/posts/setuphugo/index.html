Hugo 是什麼 ?

Hugo 是一個用 Go 編寫的快速、現代的靜態網站生成器。

Note
更多的 Hugo 介紹, 請參考 Hugo 網站.

Hugo 可安裝在不同的作業系統，包括 Windows/macOS/Linux。 現在，我們將向您展示如何在 Linux 中安裝Hugo。
請按照以下步驟操作：

我們開始吧 ！

步驟 1. 安裝 Hugo

至Hugo releases 網站 並根據您的作業系統來下載對應的安裝包。

  1. 打開瀏覽器至 Hugo 網站下載安裝包。 (這邊安裝的版本是 hugo_0.93.0_Linux-64bit.deb)

  2. 驗證安裝成功會看到對應的版本號

$ hugo version

步驟 2. 建立新的專案 

$ hugo new site $專案名稱  
$ cd $專案名稱

步驟 3. 添加主題

  1. 挑選喜愛的 Hugo 主題 並下載到專案的 themes 資料夾
$ git clone $主題網址 themes/$主題名稱
  1. 複製主題的 examplesite 資料夾內的檔案至根目錄下
$ sudo cp -r themes/$主題名稱/exampleSite/* .

  1. 配置 config.toml 檔案
baseURL = "https://{你的 Github 帳號名稱}.github.io"
theme = "$主題名稱"

步驟 4. 啟動 Hugo Server

  1. 啟動 Hugo
$ hugo server -D

  1. 打開瀏覽器並輸入 : http://localhost:1313
  2. 確認網頁可以正常瀏覽後，開始打包專案並會自動產生 public 資料夾
$ hugo

步驟 5. 將專案 Push 到 Github

  1. 建立新的 Repo

  2. 將 Repo 命名為 : {你的 Github 帳號名稱}.github.io

  3. Push public 資料夾到 Github

$ cd $專案名稱/public  
$ git init  
$ git config user.email "you@example.com"  
$ git config user.name "Your Name"  
$ git commit -m "first commit"  
$ git branch -M main  
$ git remote add origin https://github.com/{你的 Github 帳號名稱}/{你的 Github 帳號名稱}.github.io.git  
$ git push -u origin main
  1. 打開瀏覽器並輸入 : https://{你的 Github 帳號名稱}.github.io

步驟 6. Have Fun

學習東西最好的方法就是實際動手操作 !
