Install Packages Common Command

March 10, 2022 One-minute read
Linux
Ubuntu • CentOS • Linux

Here are some commands that I use frequently.

Table of Contents

  • Ubuntu
  • CentOS

Ubuntu 

# Update & Upgrade
$ sudo apt update
$ sudo apt upgrade

# Ipmitool & Openipmi
$ sudo apt install ipmitool
$ sudo apt-get install openipmi

# SSH
$ sudo apt-get install openssh-server

# ifconfig
$ sudo apt install net-tools

# tree
$ sudo apt install tree

CentOS 

# Update & Upgrade
$ yum update
$ yum upgrade

# Ipmitool & Openipmi
$ yum install ipmitool -y
$ yum install OpenIPMI  ipmitool

# SSH
$ yum install openssh openssh-server

# ifconfig
$ yum install net-tools

# tree
$ yum install tree -y
