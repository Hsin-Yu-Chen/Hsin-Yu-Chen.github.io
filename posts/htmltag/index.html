Introduction

Since Hugo version 0.60, the default Markdown renderer has been changed to goldmark. In goldmark, the default is not to render raw HTML tags. So, You may has some promble like this :

If you need use html tags, please follow me:

Let’s Start !

Step 1. Modify config.toml

You don’t need install any packages. Just add below Msg in config.toml file and enable the unsafe mode :

[markup]
  [markup.goldmark]
    [markup.goldmark.extensions]
      definitionList = true
      footnote = true
      linkify = true
      strikethrough = true
      table = true
      taskList = true
      typographer = true
    [markup.goldmark.parser]
      autoHeadingID = true
      autoHeadingIDType = 'github'
      [markup.goldmark.parser.attribute]
        block = false
        title = true
    [markup.goldmark.renderer]
      hardWraps = false
      unsafe = true
      xhtml = false

Step 2. Verify Config Success

Save config.toml file, then you may see config success.
